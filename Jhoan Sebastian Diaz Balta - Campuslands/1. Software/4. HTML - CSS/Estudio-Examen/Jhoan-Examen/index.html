Estudio de usuarios para la plataforma virtual de Campus

Este análisis busca identificar las necesidades clave de los usuarios en el uso de la plataforma virtual de Campus para formación, certificación y seguimiento.

Daniel

"Quiero certificar mis conocimientos para conseguir empleo"

Nombre

Daniel Rojas

Atributo clave

Desempleado, autodidacta, persistente

Descripción

Daniel quiere acceder a los cursos de Campus para mejorar su hoja de vida y certificarse oficialmente en programación

Necesidades

Inscripción rápida, acceso sin restricciones, navegación clara.

Obstáculos

Falta de orientación inicial, problemas de compatibilidad del navegador.

Oportunidades

Mejorar el onboarding y crear rutas de aprendizaje visibles.

Lina

"Necesito registrar notas y evaluar avances fácilmente"

Nombre

Lina Páez

Atributo clave

Instructora, organizada, metódica

Descripción

Lina da clases virtuales y necesita herramientas eficientes para gestionar grupos y retroalimentar estudiantes.

Necesidades

Subida de tareas, foros de discusión, rubricas de evaluación.

Obstáculos

Interfaz poco intuitiva, procesos largos para cargar contenido.

Oportunidades

Simplificar el panel del instructor, integrar calendario y alertas.

María

"Estoy emprendiendo y quiero capacitar a mi equipo"

Nombre

María Castellanos

Atributo clave

Emprendedora, visionaria, gestora

Descripción

María busca cursos prácticos para que su equipo de trabajo reciba formación técnica a través de la plataforma virtual.

Necesidades

Formación por roles, seguimiento de empleados, certificación grupal.

Obstáculos

No sabe cómo matricular grupos enteros o acceder como empresa.

Oportunidades

Desarrollar una vista para organizaciones con gestión de múltiples usuarios.

  • ✓ Existen múltiples tipos de usuarios: estudiantes, instructores y gestores
  • ✓ Se requieren interfaces diferenciadas según el rol
  • ✓ La plataforma necesita mayor acompañamiento inicial

Estrategia

Reestructurar la navegación y experiencia de usuario para adaptarse al perfil del visitante, con rutas claras, tutoriales integrados y mejoras en usabilidad.
